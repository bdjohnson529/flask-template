# Flask app
This is a template flask application, with scaffolding already created.

## Deployment
Install project dependencies.
```bash
pip install -r requirements.txt
```

Run the application.
```bash
python app.py
```

Access the app on in a web browser.
```bash
http://127.0.0.1:5000/
```

## Development
Run in development:
```bash
python app.py --development
```
